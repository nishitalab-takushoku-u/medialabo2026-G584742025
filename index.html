「Webコンテンツ作成」の成果

本テーマについて

HTML、CSS、JavaScriptの基本的な構文を学び、それらを組み合わせてWebページを作成する方法を習得する。また、ページの構造や見た目の設定、ユーザー操作に応じた処理や、外部サーバとの通信を実現する基礎的な技術を身に付ける。

各授業回の学習事項

第1回 WWW と HTML

今回の授業では、基本的なHTMLの書き方について学んだ。HTMLを記述する際の基本的なルールをはじめ、見出しの作成方法、リンクの設定方法、画像の埋め込み方、説明リストの作成方法などを学習した。これらはいずれもWebページを作成する上で欠かせない要素であり、Webデザインの基礎として非常に重要な内容であった。今回学んだ知識を活用しながら、今後はより見やすく分かりやすいWebページの作成に取り組んでいきたい。

演習の成果: 演習1-4: HTMLのさまざまな要素

　

第2回 CSS による Web ページの装飾

今回の授業では、基本的なCSSの書き方について学んだ。CSSを用いることで、HTMLで記述された文章の色を変更したり、表や箇条書きを見やすく整えたりするなど、直感的に理解しやすいデザインを作成できることを学んだ。また、CSSを学習しながら試行錯誤を重ねることで、多くの人にとって見やすく使いやすいWebデザインを作成したいと感じた。今後は、利用者の視点を意識しながら、より分かりやすく魅力的なWebページの作成に取り組んでいきたい。

演習の成果 演習2-3:ウェブブラウザの紹介

第3回 JavaScript の基本

今回の授業では、JavaScriptについて学んだ。JavaScriptを用いることで、プログラムを通じてさまざまな複雑な機能を実現できることを理解した。特に、ツイートを検索・取得する機能や、GoogleマップをWebページに埋め込む機能は、Webデザインを学ぶ上で実用性が高く、活用の幅が広いと感じた。また、JavaとJavaScriptの違いについても学び、変数の型や比較演算子の使い方などに違いがあることを知った。さらに、for-of構文についても学習し、Javaの拡張for文に近い書き方であることを理解した。今後は、for-of構文を使いこなしながら、簡潔で見直しやすく、汎用性の高いJavaScriptのコードを書けるようになりたい。

演習の成果 演習3-6: オブジェクトの参照

第4回 DOM と JavaScript

今回の授業では、DOMとJavaScriptについて学んだ。DOMを利用することで、ボタンやテキストフィールドなどの要素を動的に追加したり、ユーザーの操作に応じて処理を実行したりできることが分かった。特に、ユーザーが入力した内容やボタンのクリックをもとに動作を変えられる点が興味深かった。これまでのプログラミングでは、主にコンソールへの入力やコードの書き換えによって動作を確認していたが、今回は実際のWebページ上でユーザーが操作したときの動作を考えながらプログラムを作成した。そのため、より実用的なプログラム開発に近い内容を学ぶことができた。また、自分の作成したプログラムによって画面の表示が変化する様子を確認でき、DOMを使うことでWebページにさまざまな機能を追加できることが理解できた。

演習の成果 演習4-10: 足し算ページ

第5回 イベント処理

今回の授業では、イベント処理の基本的な書き方について学んだ。これまで学習してきたJavaScriptの基本的な文法に加え、DOMを利用することでWebページ上の要素を取得・編集したり、新しく作成したりできることを理解した。また、イベント処理を用いることで、ユーザーの操作に応じてページの内容を動的に変更できる点が興味深かった。演習では、DOMを操作して要素を追加する処理を実装したが、特に `beforeend` と `afterend` の使い分けに苦戦した。しかし、HTMLの構造を確認しながら、どの要素の内側に追加するのか、あるいは外側に追加するのかを意識して考えることで、正しく記述することができた。今回の学習を通して、JavaScriptによってWebページをより柔軟に操作できることを実感した。

演習の成果 演習5-7: DOMプログラミング(2)

第6回 axiosを用いた通信処理

今回の授業では、axiosを用いた通信処理について学んだ。Web上での通信は、ブラウザからサーバへデータを要求する処理と、サーバからブラウザへデータを返す処理の2つに分けて考えられることを理解した。また、JavaScriptでは通信を開始する処理だけでなく、通信が成功した場合の処理、失敗した場合の処理、そして通信終了後に実行する処理を記述する必要があることも学んだ。通信処理を利用することで外部のデータを取得でき、Webページに新たな機能を追加できることが分かり、興味深く感じた。
